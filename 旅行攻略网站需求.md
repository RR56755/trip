# 旅行攻略网站 — 需求规格说明书

## 1. 项目概述

### 1.1 项目背景
开发一个面向国内外游客的中国旅行攻略网站，集中展示中国 10 大热门旅行景点的详细攻略信息，提供景点介绍、交通指南、门票信息、最佳旅行时间、游玩路线推荐、美食推荐、住宿建议等一站式攻略服务。

### 1.2 项目目标
- 构建一个结构清晰、视觉美观的静态攻略展示网站
- 覆盖中国 10 大热门旅行目的地，每页提供深度攻略内容
- 支持响应式布局，兼容桌面端与移动端
- 便于后续扩展更多目的地

---

## 2. 功能需求

### 2.1 页面结构

#### 2.1.1 首页（index.html）
- **网站 Banner**：大图轮播或静态头图，展示中国旅行主题视觉
- **10 大景点入口**：以卡片网格形式展示每个景点，包含：
  - 景点缩略图（或占位图）
  - 景点名称
  - 一句话简介
  - 所在省份/城市
  - 热度标签（如 ⭐ 评分、推荐指数）
- **快速筛选/分类**：按地区（华北、华东、华南、西南、西北等）、类型（自然风光、历史文化、都市休闲）进行筛选
- **页脚**：版权信息、关于我们、联系方式

#### 2.1.2 景点详情页（spots/[id].html）
每个景点拥有独立的详情页面，包含以下模块：

| 模块 | 内容说明 |
|------|---------|
| **顶部头图** | 景点大图 + 景点名称 + 标签 |
| **景点简介** | 地理位置、历史背景、整体介绍（300~500 字） |
| **核心数据** | 占地面积、海拔高度、开放时间、建议游玩时长、最佳季节 |
| **门票信息** | 淡旺季价格、学生/老人优惠政策、购票方式（线上/线下） |
| **交通指南** | 外部交通（飞机/高铁/自驾）、内部交通（景区巴士/索道/步行） |
| **游玩路线推荐** | 一日游路线、两日游路线、精华游路线 |
| **必打卡景点** | 景点内部子景点列表 + 推荐理由 + 最佳拍照点 |
| **美食推荐** | 当地特色美食、推荐餐厅、人均消费参考 |
| **住宿建议** | 区域推荐（景区附近/市中心）、价格区间、酒店类型 |
| **实用贴士** | 注意事项、穿衣建议、避坑指南、紧急联系方式 |
| **旅行日程表** | 可打印/参考的日行程规划模板 |
| **互动地图** | 嵌入地图或静态示意图标注景点位置 |

### 2.2 功能特性

#### 2.2.1 导航系统
- 顶部固定导航栏：Logo、景点分类、关于我们、搜索框
- 面包屑导航：首页 > 景点分类 > 景点名称
- 侧边锚点导航（详情页）：点击快速跳转到各模块
- 底部导航：返回顶部按钮

#### 2.2.2 搜索功能
- 支持按景点名称、城市、关键词搜索
- 实时搜索建议（前端实现）
- 搜索结果高亮

#### 2.2.3 收藏与分享
- 每个景点提供收藏按钮（本地存储实现）
- 一键复制链接分享
- 社交媒体分享（微信、微博等）

#### 2.2.4 评论与评分（可选，v2 版本）
- 用户评论功能
- 评分系统（1~5 星）
- 评论排序（最新/最热）

## 4. 技术方案

### 4.1 技术栈
| 层面 | 技术选型 | 说明 |
|------|---------|------|
| **前端框架** | 原生 HTML5 + CSS3 + JavaScript (ES6+) | 轻量无依赖，便于快速部署 |
| **样式方案** | CSS Flexbox / Grid + 自定义 CSS 变量 | 响应式布局，主题统一 |
| **图标库** | Font Awesome 或 纯 CSS 图标 | 装饰性图标 |
| **字体** | Google Fonts（Noto Sans SC） | 中文字体优化 |
| **地图** | 高德/百度地图静态图或 OpenStreetMap iframe | 位置示意 |
| **图片** | Unsplash / 本地占位图 | 景点配图 |
| **部署** | GitHub Pages | 免费静态托管 |

## 5. 内容策略

### 5.1 数据来源
- 各大景点官方网站
- 中国国家旅游局发布信息
- 马蜂窝、携程等旅行平台公开攻略
- 维基百科（景点历史背景）

### 5.2 内容质量标准
- 每个景点攻略字数 ≥ 2000 字
- 所有数据（门票、开放时间等）标注"仅供参考，以景区实际公告为准"
- 图片使用免费商用授权或无版权图片
- 信息更新频率建议：每季度审查一次

## 6. 非功能需求

| 需求类型 | 具体要求 |
|---------|---------|
| **性能** | 首页加载时间 < 3s，Lighthouse 评分 ≥ 85 |
| **SEO** | 语义化 HTML 标签、合理 meta 标签、结构化数据 |
| **可访问性** | WCAG 2.1 AA 标准，支持键盘导航、屏幕阅读器 |
| **兼容性** | Chrome / Firefox / Safari / Edge 最新两个大版本 |
| **安全性** | 无外部脚本注入，纯静态站点无 XSS 风险 |
| **可维护性** | 组件化 CSS 类名、统一数据源 JSON、注释完善 |
